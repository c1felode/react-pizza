This is a Simple Prototype of pizzeria called React Pizza Here you'll find a list of pizzas and categories. You can also choose by size and type. Also a cart where you will find your choosed pizzas.

Tech-stack used:

    React
    React Router V6
    React Redux + Redux Toolkit
    Axios + Fetch
    In the future
    Typescript
    Lodash.debounce
